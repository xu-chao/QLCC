## 搭建开发环境

请先根据实际情况自行调整`vite.config.js`

```bash
git clone https://github.com/synctv-org/synctv-web
cd synctv-web
npm i
npm run dev
# http://localhost:8085
```

## 手动构建

```bash
git clone https://github.com/synctv-org/synctv-web
cd synctv-web
npm i
npm run build
```
