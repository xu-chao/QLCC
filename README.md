## Development Environment

Please adjust the `vite.config.js` according to the actual situation first

```bash
git clone https://github.com/synctv-org/synctv-web
cd synctv-web
npm i
npm run dev
# http://localhost:8085
```

## Build

```bash
git clone https://github.com/synctv-org/synctv-web
cd synctv-web
npm i
npm run build
```
